# DeepSeek 项目上下文与多文件管理助手

## 功能描述

解决 DeepSeek 网页版使用痛点：

1. **自动注入全局提示词** - 类似 `/init` 命令，每次问答自动带上你定义的上下文
2. **多文件项目管理与上下文注入** - 在浏览器内管理项目文件，提问时自动将文件结构和代码注入到对话中
3. **代码块一键复制** - 为 AI 回复的代码块添加复制按钮

## 安装方法

1. 确保浏览器已安装 [Tampermonkey](https://www.tampermonkey.net/) 扩展
2. 点击 [脚本安装链接](https://github.com/your-username/tampermonkey-tool-exts/raw/main/scripts/deepseek-project-context-helper.user.js)
3. 在 Tampermonkey 确认页面点击「安装」

## 使用说明

### 全局上下文面板

- 在面板中输入你希望每次问答都自动带上的提示词
- 例如：`你是一个资深全栈工程师。请始终遵循最佳实践，代码需简洁、健壮。`
- 内容会自动保存，刷新页面后依然有效

### 多文件项目管理

1. **新建文件** - 在输入框中输入文件名（如 `popup.html`），点击「新建」
2. **编辑代码** - 点击文件列表中的文件名，在编辑器中编写代码
3. **自动保存** - 所有修改会自动保存到 Tampermonkey 本地存储
4. **删除文件** - 选中文件后点击「删除当前文件」

### 上下文注入

当你点击 DeepSeek 的发送按钮时，脚本会自动将以下内容注入到输入框：

```
【系统上下文】
{你的全局提示词}

【当前项目文件结构】
### 文件: filename1 ###
{文件1内容}

### 文件: filename2 ###
{文件2内容}

请基于以上项目结构和上下文，回答我的问题：

{你的实际提问}
```
