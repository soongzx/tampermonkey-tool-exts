# DeepSeek 助手增强面板 用户文档

## 概述
**DeepSeek 助手增强面板** 是一个 Tampermonkey 用户脚本，为 DeepSeek 聊天页面添加一个悬浮控制面板，提供**常用命令快速插入**、**代码块列表与快速定位**、**阅读进度条**等实用功能，显著提升您在 DeepSeek 中的互动效率。

## 功能特性
### 1. 常用命令管理
- 添加、编辑、删除自定义命令（名称 + 内容）
- 点击命令名称，自动将预设内容**填充到 DeepSeek 输入框**
- 默认提供一条示例命令："代码不写注释"，内容为"请勿添加任何注释，直接给出纯净代码。"
- 命令数据存储在浏览器 `localStorage` 中，刷新页面不丢失

### 2. 代码块列表
- 自动扫描当前页面中所有代码块（`<pre><code>` 等）
- 以列表形式展示每个代码块的**编程语言**和**内容预览**
- 点击任意代码块条目，页面自动**平滑滚动并高亮**该代码块位置
- 页面内容变化（如新消息）时，列表自动更新（仅当代码块标签激活时）

### 3. 阅读进度条
- 页面顶部显示一个**渐变色进度条**，实时反映滚动阅读进度
- 始终可见，无需开关，轻量美观

### 4. 数据导入/导出
- 将自定义命令导出为 JSON 文件，方便备份或迁移
- 可导入之前导出的 JSON 文件，恢复命令列表

### 5. 悬浮面板控制
- 面板可**拖拽移动**，支持最小化/关闭
- 两个标签页："常用命令"与"代码块列表"
- 关闭后可通过刷新页面重新显示

## 安装方法
1. 确保浏览器已安装 **Tampermonkey** 扩展（[Chrome 应用商店](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo) / [Firefox 附加组件](https://addons.mozilla.org/firefox/addon/tampermonkey/)）。
2. 点击 Tampermonkey 图标 → "添加新脚本"。
3. 将提供的脚本代码完整粘贴到编辑器中，保存（Ctrl+S）。
4. 打开 [DeepSeek 聊天页面](https://chat.deepseek.com/) ，脚本自动生效。

## 使用说明
### 打开/关闭面板
- 面板默认显示在页面右下角（可拖拽移动位置）。
- 点击面板标题栏的 **─** 按钮可隐藏内容区域（仅保留标题栏）；再次点击展开。
- 点击 **✕** 按钮关闭整个面板；刷新页面后重新出现。

### 管理常用命令
1. 确保当前标签页为 **"常用命令"**。
2. **添加命令**：在下方输入框中填写"命令名称"和"命令内容"，点击"添加命令"。
3. **编辑命令**：点击命令右侧的"编辑"按钮，修改名称或内容。
4. **删除命令**：点击"删除"按钮。
5. **使用命令**：直接点击命令名称，命令内容会自动填入 DeepSeek 输入框，您可点击发送按钮或继续编辑。

### 定位代码块
1. 切换到 **"代码块列表"** 标签。
2. 页面中所有代码块将以列表形式展示（语言 + 内容预览）。
3. 点击任意代码块条目，页面自动滚动至该代码块位置，并短暂高亮显示。

### 导出/导入数据
- **导出**：点击"导出数据"按钮，浏览器将下载一个 `deepseek_assistant_backup_*.json` 文件，包含所有自定义命令。
- **导入**：点击"导入数据"按钮，选择您之前导出的 JSON 文件，即可恢复命令列表。

## 注意事项
- 脚本仅作用于 `https://chat.deepseek.com/*` 域名，不会影响其他网站。
- 常用命令数据保存在浏览器的 `localStorage` 中，清除浏览器缓存可能丢失命令，建议定期导出备份。
- 若点击命令后输入框未填充，请确认页面已完全加载，并检查控制台是否有错误信息（F12 → Console）。通常等待几秒后重试即可。
- 代码块列表会在您滚动页面或新消息到达时自动刷新，若未及时更新可手动切换标签页强制刷新。
- 阅读进度条不可关闭，若觉得干扰可自行修改脚本样式（隐藏 `#ds-reading-progress`）。

## 常见问题
**Q: 面板不显示怎么办？**  
A: 确认 Tampermonkey 脚本已启用，并强制刷新页面（Ctrl+F5）。若依然无效，尝试禁用其他可能冲突的扩展。

**Q: 命令无法自动插入输入框？**  
A: 可能是 DeepSeek 更新了页面结构。请按 F12 打开控制台，找到 `[DeepSeek助手]` 相关日志并反馈给脚本作者。临时方案：手动复制命令内容粘贴。

**Q: 代码块列表为空？**  
A: 当前页面确实没有代码块（如纯文本对话）。发送一段包含代码的消息后，列表会自动出现。

**Q: 如何修改进度条颜色或高度？**  
A: 在脚本中搜索 `#ds-reading-progress` 样式，调整 `height` 和 `background` 值即可。
